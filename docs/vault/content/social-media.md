## Strategie LinkedIn

### Page entreprise

| Element | Detail |
|---------|--------|
| Frequence | 3 posts / semaine (Lun, Mer, Ven) |
| Formats | Carousels, posts texte, articles, videos courtes, sondages |
| Ton | Thought leadership, ref [[content/brand-voice]] |
| CTA | Vers blog, demo, webinaire, whitepaper |

### Thought leadership executif

- **CEO** : Vision strategique, tendances finance, culture d'entreprise
- **CFO / Head of Product** : Insights produit, retours d'experience, use cases
- Posts personnels des dirigeants : 1-2x / semaine, authentiques, orientes valeur
- Employee advocacy : encourager les equipes a partager et commenter

### Types de contenu LinkedIn

| Pilier de contenu | Exemple | Frequence |
|-------------------|---------|-----------|
| Insights sectoriels | Tendances finance, reglementations, benchmarks | 1x / semaine |
| Fonctionnalites produit | Tips, nouveautes, captures d'ecran, mini-demos | 1x / semaine |
| Success stories clients | Temoignages, chiffres, quotes | 2x / mois |
| Culture d'equipe | Coulisses, recrutement, valeurs | 2x / mois |
| Tips finance | Conseils pratiques pour CFOs/DAFs | 1x / semaine |
