## Phase 2 -- Operationnel (En cours)

- [ ] Configurer tous les secrets GitHub (Google Ads, Gmail, Firecrawl, HubSpot)
- [ ] Premier run Iris reel (email digest)
- [ ] Premier run Scout reel (veille web)
- [ ] Premier run Nexus reel (audit Google Ads avec vrai compte)
- [ ] Premier pipeline Lead Gen complet (Scout -> Aria -> Iris)
- [ ] Health Check automatise (cron quotidien)
- [ ] Slack notifications actives

## Phase 3 -- Scale & Autonomie

- [ ] Agents capables de se declencher mutuellement sans intervention humaine
- [ ] Gemini integration pour analyse de gros contextes (> 50KB)
- [ ] Dashboard : filtres avances, historique 30 jours, metriques cumulees
- [ ] API REST pour declencher des agents depuis l'exterieur
- [ ] Obsidian Publish ou equivalent pour partager le vault en lecture
- [ ] Budget tracking par agent (tokens consommes, cout estime)

## Phase 4 -- Intelligence

- [ ] Sage capable de reecrire des prompts agents de maniere autonome
- [ ] Cross-agent learning (un agent apprend des erreurs d'un autre)
- [ ] Predictive scheduling (Ralph predit quand lancer quel agent)
- [ ] Vault auto-cleanup (archivage des notes obsoletes)
- [ ] Multi-repo support (un systeme d'agents pour plusieurs projets)
