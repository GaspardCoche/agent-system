    # Alibaba's latest AI model ran autonomously for 35 hours to optimize code for its own custom chip

![Alibaba's latest AI model ran autonomously for 35 hours to optimize code for its own custom chip](https://the-decoder.com/wp-content/uploads/2026/05/qwen3.7-max-banner.jpg)

    **Source:** [The Decoder](https://the-decoder.com/alibabas-latest-ai-model-ran-autonomously-for-35-hours-to-optimize-code-for-its-own-custom-chip/) | **Categorie:** Modeles | **Importance:** must_read

    Le modèle Qwen3.7-Max d'Alibaba a démontré des capacités d'agent autonome impressionnantes en optimisant du code pour une puce custom pendant 35 heures. Cette performance, qui surpasse Claude Opus 4.6 sur certains benchmarks, met en lumière les avancées asiatiques dans les systèmes agentiques et l'optimisation matérielle via l'IA. Les développeurs peuvent s'attendre à des outils plus sophistiqués pour l'automatisation de tâches complexes de génie logiciel et matériel.
