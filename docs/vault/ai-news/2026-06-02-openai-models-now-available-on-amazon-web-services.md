    # OpenAI models now available on Amazon Web Services

![OpenAI models now available on Amazon Web Services](https://the-decoder.com/wp-content/uploads/2026/06/aws-open-ai-amazon-news-jp-042426.jpeg)

    **Source:** [The Decoder](https://the-decoder.com/openai-models-now-available-on-amazon-web-services/) | **Categorie:** Outils & Plateformes | **Importance:** must_read

    Les modèles d'OpenAI, incluant GPT-5.5, GPT-5.4 et Codex, sont désormais disponibles via Amazon Bedrock, offrant aux entreprises un accès sécurisé et géré à travers l'écosystème AWS. Cette intégration simplifie l'adoption des modèles de pointe pour les clients AWS, permettant une construction et un déploiement plus rapides d'applications IA avec les contrôles et les flux de travail existants.
