    # Uber Caps Usage of AI Tools Like Claude Code to Manage Costs

![Uber Caps Usage of AI Tools Like Claude Code to Manage Costs](https://www.google.com/s2/favicons?domain=simonwillison.net&sz=128)

    **Source:** [Simon Willison](https://simonwillison.net/2026/Jun/3/uber-caps-usage/#atom-everything) | **Categorie:** Business & Levees | **Importance:** must_read (top story)

    La décision d'Uber de plafonner à 1 500 dollars par mois et par outil l'utilisation des agents de codage IA, comme Claude Code, est un signal fort pour les entreprises. Elle révèle la rapidité avec laquelle les dépenses en jetons peuvent s'envoler, dépassant largement les budgets prévus pour 2026 établis avant l'explosion de l'adoption de ces outils. Pour les leaders tech, cela souligne la nécessité d'une gouvernance stricte et de mécanismes de suivi des coûts pour les services d'IA générative. Cela suggère également que la valeur perçue de ces outils est significative, justifiant des plafonds élevés, mais que l'optimisation des coûts et la gestion de la consommation deviendront des compétences critiques. Les développeurs devront être plus conscients de l'efficience de leurs prompts, ce qui pourrait modifier les pratiques d'ingénierie.
