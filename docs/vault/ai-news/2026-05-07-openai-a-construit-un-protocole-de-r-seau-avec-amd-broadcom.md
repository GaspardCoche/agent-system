    # OpenAI a construit un protocole de réseau avec AMD, Broadcom, Intel, Microsoft et NVIDIA pour résoudre les goulots d'étranglement des supercalculateurs IA

![OpenAI a construit un protocole de réseau avec AMD, Broadcom, Intel, Microsoft et NVIDIA pour résoudre les goulots d'étranglement des supercalculateurs IA](https://the-decoder.com/wp-content/uploads/2026/02/openai_wall_statistics_benchmarks.png)

    **Source:** [The Decoder](https://the-decoder.com/openai-built-a-networking-protocol-with-amd-broadcom-intel-microsoft-and-nvidia-to-fix-ai-supercomputer-bottlenecks/) | **Categorie:** Outils & Plateformes | **Importance:** must_read

    OpenAI, en collaboration avec les géants de l'industrie, a développé MRC, un protocole réseau open source pour optimiser la communication entre GPU dans les supercalculateurs IA. Ce protocole réduit les couches de commutation et connecte plus de 100 000 GPU, diminuant ainsi la consommation d'énergie et les coûts. Pour les entreprises exploitant des infrastructures IA, cela signifie des performances et une efficacité accrues pour le déploiement de modèles à grande échelle.
