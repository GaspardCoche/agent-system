    # OpenAI Adds Chrome Extension to Codex, Letting Its AI Agent Access LinkedIn, Salesforce, Gmail, and Internal Tools via Signed-In Sessions

![OpenAI Adds Chrome Extension to Codex, Letting Its AI Agent Access LinkedIn, Salesforce, Gmail, and Internal Tools via Signed-In Sessions](https://www.marktechpost.com/wp-content/uploads/2026/05/blog11-2.png)

    **Source:** [MarkTechPost](https://www.marktechpost.com/2026/05/08/openai-adds-chrome-extension-to-codex-letting-its-ai-agent-access-linkedin-salesforce-gmail-and-internal-tools-via-signed-in-sessions/) | **Categorie:** Outils & Plateformes | **Importance:** must_read

    OpenAI a doté son agent de codage Codex d'une extension Chrome, lui permettant d'interagir directement avec des applications web comme LinkedIn, Salesforce et Gmail via des sessions connectées. Cette capacité transforme Codex en un agent capable d'automatiser des flux de travail complexes directement dans le navigateur, ouvrant la voie à des automatisations d'entreprise plus profondes et à des assistants personnalisés.
