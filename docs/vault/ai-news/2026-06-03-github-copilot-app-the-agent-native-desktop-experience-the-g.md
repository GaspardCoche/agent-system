    # GitHub Copilot app: The agent-native desktop experience - The GitHub Blog

    **Source:** [GitHub Blog](https://github.blog/news-insights/product-news/github-copilot-app-the-agent-native-desktop-experience/) | **Categorie:** Outils & Plateformes | **Importance:** must_read

    GitHub a lancé une application de bureau Copilot axée sur les agents, offrant une expérience native pour l'intégration des agents IA directement dans les flux de travail des développeurs. Cette initiative vise à rendre les agents IA plus accessibles et performants, en ligne avec la vision de Microsoft d'une assistance IA omniprésente dans l'environnement de développement.
