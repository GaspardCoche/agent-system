    # Microsoft caught sneaking "Co-Authored-by Copilot" into VS Code commits - even with AI off

![Microsoft caught sneaking "Co-Authored-by Copilot" into VS Code commits - even with AI off](https://the-decoder.com/wp-content/uploads/2026/05/microsoft_logo.png)

    **Source:** [The Decoder](https://the-decoder.com/co-pilot-becomes-a-co-author-in-vs-code-without-being-asked/) | **Categorie:** Outils & Plateformes | **Importance:** must_read (top story)

    Microsoft a discrètement ajouté la ligne "Co-Authored-by Copilot" dans les commits Git de Visual Studio Code, même pour les développeurs ayant explicitement désactivé les fonctionnalités d'IA. Cette pratique, qui s'apparente à une "révolution silencieuse" sous le capot, soulève des questions fondamentales sur la transparence, le contrôle et la propriété intellectuelle dans les outils de développement assistés par l'IA. Pour les B2B executives, cela implique une réévaluation urgente des politiques de gouvernance du code, de la conformité aux licences logicielles et de la confiance des équipes de développement dans l'écosystème MCP de Microsoft. Ignorer ces implications pourrait entraîner des risques juridiques et opérationnels importants à moyen terme.
