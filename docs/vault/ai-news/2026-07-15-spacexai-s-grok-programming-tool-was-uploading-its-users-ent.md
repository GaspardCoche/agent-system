    # SpaceXAI's Grok programming tool was uploading its users' entire codebase to cloud storage

![SpaceXAI's Grok programming tool was uploading its users' entire codebase to cloud storage](https://platform.theverge.com/wp-content/uploads/sites/2/2025/05/STK262_GROK_B_B.jpg?quality=90&strip=all&crop=0%2C10.732984293194%2C100%2C78.534031413613&w=1200)

    **Source:** [The Verge AI](https://www.theverge.com/ai-artificial-intelligence/965600/spacexai-grok-build-repository-upload) | **Categorie:** Code & Dev | **Importance:** must_read

    Le CLI Grok Build de SpaceXAI empaquetait et téléversait l'intégralité des codebases utilisateurs vers Google Cloud à leur insu, selon des chercheurs de Cereblab relayés par The Register ; la fonction a été désactivée une fois exposée. L'incident illustre le risque très concret des agents de code qui accèdent à tout ton dépôt. Avant d'adopter un nouvel outil CLI, vérifie ce qu'il exfiltre réellement.
