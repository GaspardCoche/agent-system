# Copilot vs. raw API access: What are you actually paying for?

**Source:** [GitHub Blog](https://github.blog/ai-and-ml/github-copilot/copilot-vs-raw-api-access-what-are-you-actually-paying-for/) | **Categorie:** Code & Dev | **Importance:** must_read

GitHub Copilot facture désormais l'usage aux tarifs API affichés des modèles, ce qui rend enfin comparable le coût de l'accès direct à l'API et celui du harnais de codage complet (workflow, policy, contexte) autour. L'article détaille ce que tu paies réellement au-delà des tokens : l'orchestration, la gouvernance et l'intégration. Un repère utile pour arbitrer entre appeler un modèle en direct et passer par un agent packagé.
