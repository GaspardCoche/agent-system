# Comment j'ai piégé Claude pour lui faire fuiter tes secrets les plus intimes

**Source:** [Simon Willison](https://simonwillison.net/2026/Jul/15/claude-web-fetch-exfiltration/#atom-everything) | **Categorie:** Regulation & Ethique | **Importance:** must_read

Le design de l'outil web_fetch de Claude est réputé résistant à l'exfiltration de données, mais Ayush Paul a trouvé une faille dans ce garde-fou. La démonstration illustre à quel point la 'lethal trifecta' (accès aux données privées + contenu non fiable + capacité d'exfiltration) reste le talon d'Achille des assistants agentiques. Un rappel concret que même les protections les mieux pensées doivent être stress-testées avant qu'on ne branche un agent sur des données sensibles.
