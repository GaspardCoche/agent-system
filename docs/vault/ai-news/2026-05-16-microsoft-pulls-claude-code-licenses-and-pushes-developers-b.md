    # Microsoft pulls Claude Code licenses and pushes developers back toward its own AI tool

![Microsoft pulls Claude Code licenses and pushes developers back toward its own AI tool](https://the-decoder.com/wp-content/uploads/2026/01/copilot_logo_anthropic_claude-1.png)

    **Source:** [The Decoder](https://the-decoder.com/microsoft-pulls-claude-code-licenses-and-pushes-developers-back-toward-its-own-ai-tool/) | **Categorie:** Outils & Plateformes | **Importance:** must_read

    Microsoft retire les licences Claude Code pour ses développeurs, les orientant vers GitHub Copilot CLI. Cette décision stratégique vise à consolider l'écosystème de développement IA de Microsoft, impactant directement les développeurs utilisant Claude Code et soulignant l'importance des stratégies de plateforme fermées.
