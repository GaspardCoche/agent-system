    # Claude Code now has a built-in browser that lets the AI read, click, and type on external websites

![Claude Code now has a built-in browser that lets the AI read, click, and type on external websites](https://the-decoder.com/wp-content/uploads/2026/07/claude_medicine.png)

    **Source:** [The Decoder](https://the-decoder.com/claude-code-now-has-a-built-in-browser-that-lets-the-ai-read-click-and-type-on-external-websites/) | **Categorie:** Outils & Plateformes | **Importance:** must_read (top story)

    Claude Code intègre désormais un navigateur natif : l'agent peut ouvrir des pages, lire leur contenu, cliquer et saisir du texte directement depuis l'environnement de développement. Les actions d'écriture sur des sites externes sont filtrées par des classifieurs, et les achats ou créations de compte exigent une validation humaine. Concrètement, un agent peut enchaîner recherche web, remplissage de formulaires et vérification de résultats dans une seule boucle, sans dépendre d'un MCP Playwright externe. Pour des workflows de prospection ou d'enrichissement, cela rapproche l'automatisation end-to-end — naviguer, extraire, qualifier — d'une primitive intégrée. Signal fort : le navigateur devient une brique native de l'agent de code, plus un add-on optionnel.
