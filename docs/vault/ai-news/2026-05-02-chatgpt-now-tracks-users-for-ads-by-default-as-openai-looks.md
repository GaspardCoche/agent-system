    # ChatGPT now tracks users for ads by default as OpenAI looks for new revenue

![ChatGPT now tracks users for ads by default as OpenAI looks for new revenue](https://the-decoder.com/wp-content/uploads/2026/05/openai_chatgpt.png)

    **Source:** [The Decoder](https://the-decoder.com/chatgpt-now-tracks-users-for-ads-by-default-as-openai-looks-for-new-revenue/) | **Categorie:** Business & Levees | **Importance:** must_read (top story)

    OpenAI a activé par défaut le suivi des utilisateurs de ChatGPT gratuit à des fins publicitaires dans les pays où les publicités sont diffusées. Cette décision marque un tournant stratégique majeur pour l'entreprise, signalant une diversification de ses sources de revenus au-delà des abonnements payants. Pour les dirigeants B2B, cela indique une pression croissante pour monétiser l'IA grand public et potentiellement l'arrivée de modèles similaires pour les services d'IA d'entreprise. Cela soulève également des questions importantes sur la protection de la vie privée et la conformité réglementaire (GDPR, CCPA) que les entreprises devront surveiller attentivement si leurs propres applications d'IA adoptent des stratégies similaires.
