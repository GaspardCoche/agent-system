    # L'App Store pour robots est arrivé : Hugging Face lance l'App Store open source Reachy Mini avec 200 applications

![L'App Store pour robots est arrivé : Hugging Face lance l'App Store open source Reachy Mini avec 200 applications](https://www.google.com/s2/favicons?domain=venturebeat.com&sz=128)

    **Source:** [VentureBeat AI](https://venturebeat.com/technology/the-app-store-for-robots-has-arrived-hugging-face-launches-open-source-reachy-mini-app-store-with-200-apps) | **Categorie:** Open Source | **Importance:** must_read

    Hugging Face a lancé l'App Store open source Reachy Mini, offrant une bibliothèque de plus de 200 applications gratuites pour les robots humanoïdes Reachy Mini. Cette initiative marque une étape importante vers la démocratisation de la robotique agentique, permettant aux développeurs de partager et d'utiliser des applications robotiques prêtes à l'emploi. C'est une révolution silencieuse pour l'intégration de l'IA et de la robotique dans des scénarios pratiques et éducatifs.
