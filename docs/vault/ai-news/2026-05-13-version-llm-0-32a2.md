    # Version : llm 0.32a2

![Version : llm 0.32a2](https://www.google.com/s2/favicons?domain=simonwillison.net&sz=128)

    **Source:** [Simon Willison](https://simonwillison.net/2026/May/12/llm/#atom-everything) | **Categorie:** Outils & Plateformes | **Importance:** must_read

    La nouvelle version alpha de llm, un outil CLI pour interagir avec les LLM, passe la plupart des modèles OpenAI à l'endpoint `/v1/responses`, permettant un raisonnement entrelacé sur les appels d'outils pour les modèles de classe GPT-5. Cela représente un bond significatif pour les développeurs d'agents IA, offrant des capacités de chaînage d'outils et de raisonnement plus sophistiquées.
