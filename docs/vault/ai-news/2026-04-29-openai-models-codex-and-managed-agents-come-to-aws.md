    # OpenAI models, Codex, and Managed Agents come to AWS

![OpenAI models, Codex, and Managed Agents come to AWS](https://www.google.com/s2/favicons?domain=openai.com&sz=128)

    **Source:** [OpenAI Blog](https://openai.com/index/openai-on-aws) | **Categorie:** Outils & Plateformes | **Importance:** must_read (top story)

    La disponibilité directe des modèles GPT d'OpenAI, de Codex et des Agents Gérés sur AWS représente un changement stratégique majeur dans le paysage de l'IA d'entreprise. Alors que Microsoft Azure bénéficiait jusqu'à présent d'une position quasi exclusive pour le déploiement de ces modèles, cette nouvelle offre sur AWS procure aux entreprises une flexibilité accrue et atténue les risques de dépendance vis-à-vis d'un fournisseur unique. Pour les développeurs, cela signifie un accès simplifié aux modèles avancés et aux outils d'agents directement dans leurs environnements AWS habituels, accélérant ainsi le prototypage et le déploiement d'applications basées sur l'IA. Cette initiative intensifie la concurrence entre les géants du cloud et pousse Microsoft à affiner sa proposition de valeur différenciée autour d'OpenAI.
