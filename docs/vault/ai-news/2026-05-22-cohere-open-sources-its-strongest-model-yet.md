    # Cohere open-sources its strongest model yet

![Cohere open-sources its strongest model yet](https://the-decoder.com/wp-content/uploads/2026/05/Cohere-Command-A-Plus-3840x1920-1.webp)

    **Source:** [The Decoder](https://the-decoder.com/cohere-open-sources-its-strongest-model-yet/) | **Categorie:** Open Source | **Importance:** must_read

    Cohere a rendu open source Command A+, son modèle linguistique le plus puissant à ce jour, sous licence Apache 2.0. Ce modèle se distingue par sa capacité à intégrer des citations natives, reliant chaque affirmation factuelle à sa source d'origine, et par des techniques de quantification sans perte. Cela offre aux développeurs des outils robustes pour construire des applications d'IA fiables, avec une transparence et une efficacité accrues.
