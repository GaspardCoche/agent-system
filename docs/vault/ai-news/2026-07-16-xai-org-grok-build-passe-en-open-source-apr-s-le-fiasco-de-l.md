# xai-org/grok-build passe en open source (après le fiasco de l'upload de répertoires)

**Source:** [Simon Willison](https://simonwillison.net/2026/Jul/15/grok-build/#atom-everything) | **Categorie:** Open Source | **Importance:** must_read

L'outil CLI de xAI a été open-sourcé, quelques jours après un backlash violent : lancé dans un dossier, il pouvait uploader tout le répertoire vers les buckets Google Cloud de xAI — un utilisateur l'a vu partir depuis son home directory. L'ouverture du code permet enfin d'auditer ce que fait réellement l'agent de coding. Une leçon crue sur la confiance qu'on accorde aux agents CLI qui touchent au système de fichiers.
